# Notes

## vs environment

### include
两种方式：
* 属性 -> VC++目录 -> 包含目录  //此方式影响环境变量IncludePath
* 属性 -> C/C++ -> 常规 -> 附加包含目录

### lib
1. 直接添加到项目中：添加 -> 现有项
2. 添加库目录，两种方式：
  * 属性 -> VC++目录 -> 库目录  //此方式影响环境变量LibraryPath
  * 属性 -> 连接器 -> 常规 -> 附加库目录 
3. 指定具体依赖库文件：属性 -> 连接器 -> 输入 -> 附加依赖项

### dll
两种方式：
* 属性 -> 调试 -> 环境：path=DirectoryOfDLL;  //影响环境变量PATH
* 将dll拷贝到exe目录下
